# SkyWalker541 Overlays
**Overlays and bezels for the Retroid Pocket Classic, Ayaneo Pocket DMG, and Retroid Pocket Mini V2**

---

## ✨ Also Check Out: PT SkyWalker541 Shader

> **Want to take your emulation further?** Pair these overlays with my **[PT SkyWalker541 Pixel Transparency Shader](https://github.com/SkyWalker541/PT-SkyWalker541)** — a performance-focused GLSL shader built for low-power handhelds.
>
> It restores the look of original GB, GBC, and GBA hardware by detecting white and bright pixels and blending them toward a procedurally generated backing texture — simulating the translucent screen material that game developers originally designed around. It also includes Grid, LCD Dot, and CRT Phosphor display simulation modes.
>
> **Best paired with bezel-only overlays (no grid).** Using the shader with grid overlays will result in a double grid effect.
>
> **Compatible with RetroArch and NextUI. [→ Get it here](https://github.com/SkyWalker541/PT-SkyWalker541)**

## Setup

> **Tip:** For STN and Pixel Pop overlays, turn colorization **OFF** in RetroArch settings for the most accurate representation of original hardware.

- In the **On-Screen Overlay** menu, set **Auto-Scale Overlay** to **OFF**
- Adjust overlay **Opacity** to your preference
- All overlays are fully compatible with the Ayaneo Pocket DMG and Retroid Pocket Mini V2 (same screen)
- New to overlays? See [Russ' (RetroGameCorps) installation guide](https://retrogamecorps.com/2024/09/01/guide-shaders-and-overlays-on-retro-handhelds/)

> **Note:** Bezel Fade options add a shadow to the edge of the game screen, helping recreate the look of an inset screen on older handhelds.
